## ✨ À propos

JeuxCracks Launcher est une application de bureau nouvelle génération conçue pour offrir une expérience **ultra-premium**. Oubliez les listes ennuyeuses : profitez d'une interface cinématique, fluide et immersive pour gérer et découvrir votre bibliothèque de jeux.

## 🚀 Fonctionnalités Clés

### 💎 Expérience Utilisateur Premium
*   **Interface Cinématique** : Page d'accueil immersive "Showcase" avec vidéos en arrière-plan.
*   **Design Moderne** : Effets de verre (Glassmorphism), animations fluides et typographie soignée.
*   **Catalogue Editorial** : Présentation des jeux sous forme de cartes larges et détaillées.

### ⚡ Performance & Technique
*   **Téléchargement P2P** : Gestionnaire de torrents intégré ultra-rapide (WebTorrent Hybrid).
*   **Installation Automatique** : Décompression (Unrar) multi-threadée et installation silencieuse.
*   **Détection Intelligente** : Scan automatique des exécutables pour trouver le bon lanceur de jeu.

### 🔄 Système Automatisé
*   **Mises à jour Auto** : Comme Steam ou Discord, le launcher se met à jour tout seul au démarrage.
*   **Synchronisation** : Votre bibliothèque est toujours à jour avec les derniers ajouts.

## 🛠️ Installation (Développement)

Si vous souhaitez contribuer ou compiler le projet vous-même :

```bash
# Cloner le dépôt
git clone https://github.com/Wasmata/jeuxcracks_app.git

# Installer les dépendances
npm install

# Lancer en mode développement
npm run dev
```

## 📦 Compilation (Build)

Pour créer l'installateur Windows (.exe) :

```bash
# Nécessite un jeton GH_TOKEN pour l'auto-update
npm run build
```
